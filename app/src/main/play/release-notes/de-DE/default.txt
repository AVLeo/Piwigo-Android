This is a beta version, with major improvements but still not intended for production use. Seien Sie vorsichtig und stellen Sie sicher, dass Sie ein vollständiges Backup Ihrer Galleriedaten gemacht haben.

In dieser Version haben wir
- die Unterstützung für Android 4.0 "Ice Cream Sandwich" und 4.1 "Jelly Bean" entfernt
- das Design auf den neuen Piwigo Stil angepasst
- die Sprachen Deutsch, Französisch und eine erste Version der indischen Sprache Kannada hinzugefügt
- Unterstützung der Erstellung von Alben
- automatische Korrektur der Galerie-Seite hinzugefügt (falls möglich wird automatisch HTTPS verwendet)
